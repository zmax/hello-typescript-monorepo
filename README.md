# Hello TypeScript with Monorepo Style Project

choose which one you prefer to:

- `master` branch: main codes + packages
- `all-in-packages` branch: all codes in packages

## Install

```bash
yarn bootstrap
yarn build
```

## Develop

```bash
yarn dev
```

## Add new package

```bash
./new <scope> <name>
yarn bootstrap
```

## Test

```bash
jest
# or jest --watch
```
